# Stickman Clicker Game

Welcome to the Stickman Clicker Game! This is an engaging clicker game where players earn coins by clicking on a stickman character. The game features various upgrades and armor pieces that enhance gameplay, along with an auto-click feature for added convenience.

## Features

- **Click to Earn Coins**: Each click on the stickman earns you coins that contribute to your total gold saved.
- **Upgrades**: Purchase upgrades that improve your clicking efficiency from level 0 to level 10.
- **Armor System**: Equip armor pieces such as swords, boots, helmets, chestplates, and pants, using materials like bronze, iron, gold, and diamond. Each armor piece has multipliers of x1, x2, x3, and x5.
- **Auto-Click Feature**: Spend 2,000 coins to purchase an auto-click feature that automatically clicks for you after purchase.
- **Animations**: Watch the stickman perform a slap animation with each click, and an attack animation if equipped with a sword.
- **Visual Effects**: After clicking continuously for 60 seconds, a special PNG effect appears in the background.
- **Dynamic Scenery**: The scenery changes every 50 and 100 levels, cycling through cities, green backgrounds, houses, forests, and random landscapes.

## Getting Started

1. **Clone the Repository**: 
   ```bash
   git clone <repository-url>
   cd stickman-clicker
   ```

2. **Install Dependencies**: 
   ```bash
   npm install
   ```

3. **Run the Game**: 
   ```bash
   npm run dev
   ```

4. **Open in Browser**: Navigate to `http://localhost:3000` (or the specified port) to start playing!

## Gameplay Instructions

- Click on the stickman to earn coins.
- Use the coins to purchase upgrades and armor from the Upgrade Panel.
- Monitor your total gold and upgrades in the HUD.
- Enjoy the dynamic scenery as you progress through levels!
